# Tarot Psychology Consulting Web App - PRD

## 1. 프로젝트 개요 (Overview)
- **목표:** '1,000명의 진정한 팬' 확보 및 인스타그램 바이럴 성장에 초점을 맞춘 타로 심리 상담 웹앱 MVP 구축.
- **핵심 전략:** 실시간 AI 생성이 아닌 사전 구축된 DB 기반의 난수 추출 방식 적용으로 초기 개발 리소스 최소화. 유료 비즈니스 모델은 '컨시어지(Concierge) MVP' 형태로 빠르게 시장 검증.

## 2. 사용자 스토리보드 (Storyboard)
### 2.1. 인입 및 1장 뽑기 (무료 핵심 경험)
1. **진입:** 사용자가 지인의 인스타그램 스토리 링크나 웹 검색을 통해 앱 메인 화면에 접속합니다.
2. **카드 뽑기:** 메인 화면에서 직관적인 텍스트("오늘 당신을 위한 수호 카드 1장을 뽑아보세요")와 함께 위로를 주는 아름다운 테마의 '뽑기' 버튼을 터치합니다.
3. **결과 확인:** 부드러운 애니메이션(예: 카드 뒤집기) 후 22장의 메이저 아르카나 타로 카드 중 하나가 무작위로 등장합니다. 카드 이미지와 함께 3단계의 감성 문구(공감 - 통찰 - 긍정적 확언)가 차례대로 부드럽게 출력됩니다.

### 2.2. 바이럴 루프 (인스타그램 공유)
4. **저장 및 공유:** 결과 하단의 '인스타그램 스토리에 공유하기' 버튼을 누르면, 화면이 9:16 비율의 아름다운 이미지 카드 템플릿으로 캡처/변환되어 기기에 저장되거나 곧바로 공유 창으로 연결됩니다.
5. **재유입:** 스토리를 본 다른 사용자들이 이미지 하단의 고유 워터마크나 짧은 링크(Short Link)를 통해 앱으로 새롭게 진입합니다.

### 2.3. 컨시어지 MVP (유료 플랜 전환)
6. **심층 상담 요청:** 무료 결과만으로 부족하여 더 깊은 특정 고민("상대방의 속마음은?", "나의 연애운은?") 상담을 원할 경우, 화면 하단의 '전문가에게 심층 해석 요청하기' 버튼을 클릭합니다.
7. **수동 처리(Concierge):** 복잡한 자동 결제 시스템 도입 대신, 클릭 시 카카오톡 오픈채팅방이나 이메일(mailto) 작성 창으로 바로 연결됩니다. 사용자는 직접 개발자/타로 리더에게 상황을 보내며, 계좌번호 안내 및 수동 상담 서비스를 받게 됩니다.
